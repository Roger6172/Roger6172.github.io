机器学习项目

信用卡欺诈检测系统

基于高度不平衡(欺诈样本0.172%)的欧洲信用卡交易数据(28.5万条)，系统性实施特征工程：重构Time为Hour/Day时序特征，应用LogAmount对数变换和RobustScaler标准化金额特征。设计Stacking集成学习框架，通过五折分层交叉验证生成OOF预测作为元特征，避免数据泄露。模型在测试集上达到AUC 0.9768、召回率81.63%、精确率80%、误报率0.035%。

技术栈:

Python Scikit-learn SMOTE Stacking集成 特征工程

项目成果:

  • AUC-ROC: 0.9768，召回率: 81.63%，精确率: 80%
  • 较单一模型(XGBoost/随机森林)召回率提升9-16个百分点
  • 完成模型工程化准备，实现预处理流水线和预测模型的持久化
实时流处理项目

实时评论分析系统

设计实时数据处理架构，基于Kafka+Redis构建评论分析流水线，实现从数据采集、实时处理到可视化展示的全链路部署。采用Kafka消息队列解耦系统组件，设计消费组策略与重试机制保障数据可靠传输；利用Redis Hash结构存储实时统计，List结构缓存处理记录，优化查询性能。通过Docker容器化部署Kafka和Zookeeper，解决Windows环境下大数据组件的依赖冲突与环境隔离问题。

技术栈:

Kafka Redis Docker Streamlit 实时处理

项目特点:

  • 基于Kafka+Redis构建实时数据处理流水线
  • 使用Docker容器化解决环境兼容性问题
  • 完整的流处理架构设计与实现
  • 基于Streamlit开发交互式可视化看板
